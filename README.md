# Chess By Jayden😂

A basic chess game built in Java.

## Features

- Chess board rendering
- Piece movement
- Mouse interaction
- Image-based pieces

## Download

[Download For Mac](https://github.com/cigsnredwine/chess-game-java/releases/latest)

## How to Run

### For Windows:
#### Option 1: Run the JAR

```bash
java -jar ChessGame.jar
```
#### Option 2: Run from source:
```bash
cd src
javac main/*.java piece/*.java
java main.Main
```
### For Mac:
- Install from .dmg

## Requirements
- Java JDK 17 or higher

## Notes
- Image assets are stored in res/piece/
- If images fail to load, check file paths
